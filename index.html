2. Table of IDPs (Imaging-Derived Phenotypes)

This includes names and descriptions of all 805 IDPs, categorised into sMRI (n = 373) and dMRI (n = 432) modalities. Each row links to the Manhattan plot for that IDP under each analysis group. The final columns show SNP heritability h2 and standard error for each sex-stratified and sex-combined analysis.

5. Summary Statistics Downloads

File format: Each file is a gzipped text file (.txt.gz) containing GWAS summary statistics for one IDP.
Naming convention: [group]_XXXX.txt.gz where XXXX is the 4-digit IDP number (0001–0805).
Columns: CHR   SNP   BP   A2   A1   AF   INFO   MAF   BETA   SE   T   P
